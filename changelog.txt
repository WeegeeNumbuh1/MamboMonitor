v.0.2.3 - 2026-07-06
  - Swapped 'fake-useragent' python module with its modern
    replacement 'forge-useragent'

v.0.2.2 - 2026-06-17
  - Add improved support for running on a Raspberry Pi 5
    - Requires latest version of rgbmatrix as of this Update
  - Updated rgbmatrix install script
  - Splash screen uses updated techniques used by FlightGazer v.11.3.0

v.0.2.1 - 2026-05-31
  - Depreciate and remove custom canvas sizing feature
    - RGBMatrixEmulator v0.17.0 and newer handles correct sizing with mappers

v.0.2.0 - 2026-05-20
  - Fix git tracking for missing module dependency
  - Improved gif frame duration consistency
  - More consistent gif display duration to account for processing time
  - Make canvas dimension override optional
  - GIF display time is now controlled only by the config file
  - Fix positioning for flattened transparent gifs

v.0.1.1 - 2026-05-18
  - Added gif flattening for transparent gifs
    - Eliminates perimeter jaggedness during the downsizing process

v.0.1.0 - 2026-05-14
    - Initial Release
    - Validated to work for an entire convention weekend
